## Step 1: Install Dependencies
```bash
npm install
```

## Step 2: Start the Server
```bash
npm start
```

Server will run on `http://localhost:3000`

## Step 3: Open in Browser
1. Open your browser
2. Go to `http://localhost:3000`
3. Enter a username
4. Click "Login / Create Account"

## Step 4: Play!

### For Local Game:
- Click "Local Game" mode
- Start playing!

### For Online Game:
1. Click "Friends" button
2. Search for a friend's username
3. Click "Add Friend"
4. Accept friend request
5. Click "Challenge" on your friend
6. Start playing online!

### For Voice Chat:

## Troubleshooting

### Server won't start?
- Make sure Node.js is installed: `node --version`
- Make sure port 3000 is not in use
- Try: `npm install` again

### Can't connect to server?
- Make sure server is running
- Check if URL is correct: `http://localhost:3000`
- Check browser console for errors

### Voice chat not working?
- Allow microphone permissions in browser
- Make sure you're in an online game
- Check browser console for errors

### Friend search not working?
- Make sure both users are logged in
- Check if username is correct
- Make sure server is running
